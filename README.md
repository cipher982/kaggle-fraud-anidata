Results of training with k-fold (3) on various random samplings of the training data using LGBM.

| ------------- |:-------------:|
| 0.01%         | 0.875439      |
| 0.1%          | 0.942221      |
| 1.0%          | 0.959162      |
| 10%           | 0.961657      |
